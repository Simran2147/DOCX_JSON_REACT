
# DOCX to JSON & React Display

## Overview

This project is a technical assessment aimed at extracting structured data from a Word document using Python, converting it into JSON format, and developing a React.js application to present the extracted data in a structured and user-friendly format.


## Features

1. Extracts headers, text content, and tables from a Word document.

2. Converts extracted data into a structured JSON format.

3. Displays the extracted data in a React.js web application.

4. Provides a clean and structured layout for better readability.


## Technologies Used

1. Backend: Python (for data extraction and JSON conversion)

2. Frontend: React.js (for displaying extracted data)

3. Styling: Material-UI / Basic CSS
## Setup Instructions


#### Backend (Python)

#### 1. Clone the repository:

```git clone https://github.com/Simran2147/DOCX_JSON_REACT.git```

```cd DOCX_JSON_REACT```

#### 2. Install dependencies:

```pip install -r requirements.txt```

#### 3. Configure Firebase:

```python extract_data.py```

#### Frontend (React.js)

#### 1. Navigate to the React project folder:

``` cd extract_data```

#### 2. Install dependencies

```npm install```

#### 3. Start the development server

```npm start```

#### 4. Open ```http://localhost:3000/``` in your browser.
